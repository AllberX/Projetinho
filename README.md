# Manual de Cadastro Facial

Este projeto é uma página web interativa com um manual visual e passo a passo para o cadastro facial do equipamento Intelbras SS 3532 MF.

## 📌 Sobre o projeto

A página reúne instruções claras, imagens ilustrativas e boas práticas para facilitar o processo de cadastro de usuários diretamente no equipamento.

## ✨ O que você encontra

- Guia visual do processo de cadastro
- Passo a passo com exemplos ilustrativos
- Seções com boas práticas para melhorar o reconhecimento facial

## ▶️ Como visualizar

1. Abra o arquivo `index.html` no navegador.
2. Ou utilize uma extensão como Live Server para abrir a página com mais praticidade.
